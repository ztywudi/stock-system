# 📦 库存管理系统 v2.0

**Python + SQLite 本地服务器架构**，数据存储在硬盘文件中，清浏览器缓存、换浏览器、换设备（局域网）数据都不受影响！

## ✨ 核心优势

| 对比 | 旧版（纯前端） | 新版（服务器模式） |
|------|--------------|------------------|
| 数据存储 | 浏览器 IndexedDB | SQLite / JSON 文件（硬盘） |
| 清缓存 | ❌ 数据丢失 | ✅ 不受影响 |
| 换浏览器 | ❌ 数据不互通 | ✅ 同一数据库 |
| 局域网多人 | ❌ 各自独立 | ✅ 共享数据 |
| 数据备份 | 导出 JSON | 导出 JSON + 直接复制数据文件 |

## 🚀 快速开始

> **前提：电脑需要安装 Python 3**（通常已自带，终端输入 `python --version` 确认）

### Windows
双击 `start.bat`

### Mac / Linux
```bash
cd inventory-system
python3 server.py
```

### 首次使用
1. 浏览器打开 `http://localhost:8765`
2. 默认账号：`admin` / `admin123`
3. 进入「系统设置 → 用户管理」添加用户

### 局域网共享
同一个 WiFi 下，其他设备访问 `http://你电脑的IP:8765`

## 💾 数据安全

- 数据存储在 `inventory.db`（Python版）或 `data.json`（Go版）文件中
- 直接复制数据文件即可备份
- 也可以用内置的「导入/导出」导出 JSON
- 清除浏览器缓存**不会**影响数据！
