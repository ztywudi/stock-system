# 📦 库存管理系统 v2.0 — 服务器模式

**Python + SQLite 本地服务器架构**，数据存储在硬盘 `.db` 文件中，清浏览器缓存、换浏览器、换设备（局域网）数据都不受影响！

## ✨ 核心优势

| 对比 | 旧版（纯前端） | 新版（服务器模式） |
|------|--------------|------------------|
| 数据存储 | 浏览器 IndexedDB | SQLite 文件（硬盘） |
| 清缓存 | ❌ 数据丢失 | ✅ 不受影响 |
| 换浏览器 | ❌ 数据不互通 | ✅ 同一数据库 |
| 局域网多人 | ❌ 各自独立 | ✅ 共享数据 |
| 数据备份 | 导出 JSON | 导出 JSON + 直接复制 .db 文件 |

## ✨ 功能一览

| 功能 | 说明 |
|------|------|
| 📊 **工作台** | 库存概览、库存预警提示、最近出入库动态 |
| 📋 **物品管理** | 物品的增删改查，支持搜索过滤 |
| 🏷️ **分类管理** | 自定义分类，方便物品归类 |
| 📥 **入库管理** | 入库录入，自动更新库存 |
| 📤 **出库管理** | 出库录入，自动扣减库存（库存不足时自动拦截） |
| 📜 **出入库记录** | 全部历史记录一览 |
| 🔍 **库存盘点** | 创建盘点任务，比对账面与实际库存 |
| 📈 **统计报表** | 分类统计、近30天出入库趋势、价值TOP10 |
| 🖨️ **打印单据** | 出入库单据打印 |
| 🔐 **账号登录** | 管理员/操作员/只读用户三级权限，支持多用户 |
| 👥 **用户管理** | 添加/编辑/删除用户，重置密码（管理员专用） |
| 📤 **导入导出** | JSON 备份 / CSV 导出 |
| 🌐 **局域网访问** | 同一 WiFi 下多设备共享访问 |

## 🚀 快速开始

> **前提：电脑需要安装 Python 3**（通常已自带，终端输入 `python --version` 确认）

### Windows
双击 `start.bat`

### Mac / Linux
```bash
cd inventory-system
python3 server.py
```

### 首次使用
1. 浏览器打开 `http://localhost:8765`
2. 默认账号：`admin` / `admin123`
3. 进入「系统设置 → 用户管理」添加用户

### 局域网共享
同一个 WiFi 下，其他设备访问 `http://你电脑的IP:8765`

## 📂 项目结构

```
inventory-system/
├── server.py         # ⭐ Python 服务器（启动这个）
├── index.html        # 前端页面
├── inventory.db      # ⭐ SQLite 数据文件（启动后自动生成）
├── start.bat         # Windows 启动脚本
├── start.sh          # Mac/Linux 启动脚本
├── README.md         # 说明文档
├── css/style.css     # 样式
└── js/
    ├── db.js         # API 数据层
    └── app.js        # 应用主逻辑
```

## 🔐 用户与权限

|------|----------|
| 👑 **管理员** | 全部功能 |
| 🔧 **操作员** | 物品/分类/出入库/盘点/统计/导出 |

## 💾 数据安全

- 数据存储在 `inventory.db` 文件中
- 直接复制 `inventory.db` 即可备份
- 也可以用内置的「导入/导出」导出 JSON
- 清除浏览器缓存**不会**影响数据！
